# ToDo List Front-end

### Technologies

- React;
- styled-omponents;
- react-toastify;
- react-router-dom;
- axios.

### How to use?

Install the dependencies with the following command

```
npm install
```

or

```
yarn
```

Create the file `.env` with the desired API url:

```
REACT_APP_BASE_URL=url/para/sua/api
```

Run the server with the following command

```
npm start
```

or

```
yarn start
```
